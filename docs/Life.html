Description

Simulating Conway's Game of Life on a bounded board.

-- Get 15 generations from a 30x80 random world using 5 as seed:
gens = take 15 . iterate generation $ mkRandWorld (mkStdGen 5) 30 80
-- Print each generation:
mapM_ putStrLn . map showWorld $ gens

World generation

mkWorldFrom :: [Cell] -> Height -> Width -> World Source #

mkWorldFrom v w h generates a w * h sized world grid with cell values from v. The values are repeated for all cells if the length of v is less than w * h.

mkRule :: [Int] -> [Int] -> Rule Source #

mkRule survive birth returns a new rule, where survive and birth specify cell counts for survival and birth of a cell:

mkRule [2,3] [3] == conwayRule

defaultRule :: Rule Source #

Default rule used by generation. Same as conwayRule.

conwayRule :: Rule Source #

Conway's rule (23/3): A cell is born from 3 living cells. A cell only stays alive with 2 or 3 alive neighbours.
